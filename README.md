# FuelSense-AI


## Overview
FuelSense AI is a Machine Learning-Based LPG Consumption Prediction and Demand Intelligence System.

## Features
- LPG depletion prediction
- Automatic refill booking
- Area-wise demand forecasting
- Prediction accuracy visualization

## Technology Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Langflow
- IBM watsonx AI
- Granite model

## Domain
Artificial Intelligence | Machine Learning | Predictive Analytics
